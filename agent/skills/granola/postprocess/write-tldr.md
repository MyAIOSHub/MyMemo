# Write TL;DR — 3-Sentence Share-able Recap

What's the TL;DR of this meeting? Write **three short sentences** that are appropriate for pasting over to share with my team immediately.

The first sentence should say something like "Met with [name of person/company] to discuss [main topic]."

Format:
```
Sentence 1
- Sentence 2
- Sentence 3
```
